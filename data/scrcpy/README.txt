Carpeta de scrcpy portable para el panel (servidor → descarga ZIP a cada PC del NOC).

La primera vez que alguien pulse «Descargar scrcpy», el servidor copia aquí los archivos
desde la ruta de settings.json (si esta carpeta está vacía).

También puedes copiar manualmente el contenido de scrcpy-win64 (adb.exe, scrcpy.exe, DLLs…)
directamente en esta carpeta: data/scrcpy/

No subas esta carpeta llena a git (solo este README).
